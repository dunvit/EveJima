Список изменений в обновлении 1.32


Дополнения и изменения:

+ При более чем одном залогиненном в EveJima и игре персонаже активация окна с игрой вызовет автоматический выбор пилота в EvaJima Теперь вам не нужно переключать пилотов вручную.
+ Добавлена возможность изменения размеров окна в режиме "Браузер" и "Полный экран"
+ Теперь ролики https://www.youtube.com при разворачивании в режим "Полный экран" разворачиваются в пределах окна программы EveJuma

Исправленные ошибки:

+ Исправлены размеры окна браузера при развертывании его на весь размер аппликации.
+ Исправлены проблемы многократного обновления вкладок приводящее к миганию кнопок при переключении между вкладками.

Patch notes for 1.31 Release 

Features & Changes:

+ When more than one pilot logged in EveJima and Eve Online activation game window will cause automatic selection of the pilot in EvaJima. Now you do not need to switch pilots manually.
+ Added ability to change the size of the window in the "Browser" and "Full Screen"
+ Now https://www.youtube.com movies in mode "Full Screen" takes all place within EveJuma window

Defect Fixes:

+ Fixed browser window size when deploying it to the entire size of the application.
+ Fixed problem of multiple updates tabs leading to the blinking buttons when switching between tabs.
