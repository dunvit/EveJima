Список изменений в обновлении 1.27

Благодарю человека с никнеймом kweler за предаставление адреса для Еваджимы. http://evejima.w-space.org/

Разделение на версии для х86 (стандартная) и х64 (файлы для скачивания каждой из версий внизу релиза. При самостоятельной компиляции проекта из исходников воспользуйтесь Configuration Manager в Visual Studio для создания х86 или х64 версий)

Дополнения и изменения:

+ История путешествий в сигнатурах. Сохранение текущих сигнатур насканенных более 25% прогресса.
+ В заголовок окна добавлена информация о текущем классе системы и статиках.
+ Добавление во вкладке "Pilots" навигации на ресурс "Еverate"
+ Добавление файла конфигурации (settings.txt)

Исправленные ошибки:

+ Исправлено падение программы если в момент запуска Git Repository (https://github.com/) находится в оффлайне.
+ Устранена проблема когда во вкладке "Pilots" двойной клик мыши по пустому "History" приводил к вылету аппликации.
+ Во вкладке "Browser" длина названия загруженной страницы уменьшена до 20 символов
