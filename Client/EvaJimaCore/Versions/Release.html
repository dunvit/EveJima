Hi everyone, I want to introduce to you a new version of application that can replace built-in browser and automatic navigation in W-Space. EveJima started in December 2016. Russian forum discussions: `http://forum.eve-ru.com/index.php?showtopic=116834` Old Eve forum discussions: `https://forums-archive.eveonline.com/topic/504529/` EveJima this is desktop open source project based on ESI interface and hosted in GitHub: https://github.com/dunvit/EveJima Eve Jima version 2.02: ![image|690x467](upload://5SWjauRB5I83zQu7SxbzOxewxHb.png) 1. Application use only (EVE ESI). `https://forums.eveonline.com/uploads/default/original/3X/f/3/f35fd064db4b4613f29b0bd74e46ec436debed0c.jpg` 2. Application supports unlimited amount of logged in pilots. `https://forums.eveonline.com/uploads/default/original/3X/d/a/da43f4b283865e2da5db91231fe94302040de7f1.png` 3. You can always minimize the application and leave it above other windows. `https://forums.eveonline.com/uploads/default/original/3X/7/9/79799e9babc63fcc2d577d4d68981408866cbd0f.jpg` 4. Application show you location info in 3 second after change location (wormhole entering or jump in star gate) `https://forums.eveonline.com/uploads/default/original/3X/6/0/6049d405919d137a808cc3e1253eaa00a3eb7128.jpg` 5. Once click in buttons (see image from point 4) show information about current location in imbedded browser. (ZKillbord, Ellatha and more another pre-sets) `https://forums.eveonline.com/uploads/default/original/3X/c/d/cdd5060bd958eb4f2446e8c566ff70a4902f8d40.jpg` 6. Built-in browser can save favourites and bookmarks like standard browser. `https://forums.eveonline.com/uploads/default/original/3X/8/6/8640ab5b6b3db6225daae81d52ade0e7bf5717d5.png` 7. Application gives you the ability to work with saved coordinates filtering thru old and updated signatures 7.1 Select all anomalies (crl-A) and copy (crl-C) `https://forums.eveonline.com/uploads/default/original/3X/3/8/385427258aeded62d1422aa27035e18131c6eac8.png` 7.2 Click on button "Paste Cosmic Signatures" `https://forums.eveonline.com/uploads/default/original/3X/1/d/1da9cb8f5de35e29ca1cb112592c5bca44f5e427.png` 7.3 Select all bookmarks for currents solar systems and copy (crl-C) `https://forums.eveonline.com/uploads/default/original/3X/a/7/a7321c97fe300ceae0a4ad82df03d7293b2736c8.png` 7.4 Click on button "Paste Location Bookmarks" `https://forums.eveonline.com/uploads/default/original/3X/1/0/102bcf761a5141b064e5624fcc8712b804c04e6c.png` 7.5 Click on button "Analyze". The left side shows the coordinates for deletion. On the right side, new signatures are shown. `https://forums.eveonline.com/uploads/default/original/3X/2/7/27c04e3e00f4cd5be2d312f2ebd9378911152b34.png` 8. On select cosmic anomaly and copy (crl-C) in clipboard you get rebuilded string in format: type [ID] name. You can remove this option in tab "Settings" `https://forums.eveonline.com/uploads/default/original/3X/2/9/2940c0cbb062e8f8248c1f92b8200bfdeb9badb9.png` 9. If in setting (tab "Settings") selected "Use Wormholes Map" for true then in tab "Location" you show map of your travelling. Map updated each entering to new location. You can share map for all pilots in all your accounts. `https://forums.eveonline.com/uploads/default/original/3X/2/9/2940c0cbb062e8f8248c1f92b8200bfdeb9badb9.png` 10. In tab "Pathfinder" you can get all opened information (kills npses, pilots and ships in last hour) in distance 5 jumps from pilot location. Application get names of bookmarks from folder [EveJima] in personal bookmarks and show only star system included this bookmarks. `https://forums.eveonline.com/uploads/default/original/3X/4/6/467401af0f1a57ed84b181af615ddd8ffb0ace28.png` You can download msi or open sources in GitHub releases `https://github.com/dunvit/EveJima/releases` or direction link: `https://github.com/dunvit/EveJima/releases/download/2.02/EveJimaInstaller.msi` I would be grateful for any suggestions and corrections.
