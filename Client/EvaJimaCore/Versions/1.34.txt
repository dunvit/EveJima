Список изменений в обновлении 1.34


Дополнения и изменения:

+ Добавлена вкладка "Lost and found Office" доступная из "Location". Вы можете опубликовать заявку на пропавшую солнечную систему выставив вознаграждение. Каждый пилот вошедший в такую систему получит сообщение о том, что вы разыскиваете эту систему. 
+ Добавлена иконка для программы которая отображается при закреплении программы на таскбаре.	

Исправленные ошибки:

+ Исправлена ошибка из-за которой нельзя было изменить размеры окна во вкладке "Browser"
+ Исправлена ошибка из-за которой сигнатуры из посещенной солнечной системы не загружались во вкладку "Travel History"
+ Исправлена ошибка при которой жизненный цикл червоточины при наведении мышки показывался только для первого статика.

Patch notes for 1.34 Release 

Features & Changes:

+ Added tab "Lost and found Office" available from the "Location". You can post a request for the missing Wormhole exposed reward. Each pilot went down in such a system will receive a message stating that you are looking for this wormhole.
+ Added icon for the program that is displayed when pin the program to the taskbar.

Defect Fixes:

+ Fixed a bug due to which it was impossible to change the size of the window in the "Browser" tab
+ Fixed a bug due to which the signature of the visits of the solar system is not loaded in the tab "Travel History"
+ Fixed a bug where the life cycle of the wormhole when the mouse is shown only for the first static.
