Список изменений в обновлении 1.28


Дополнения и изменения:

+ Изменен адрес репозитория с https://github.com/dunvit/WHL/releases на https://github.com/dunvit/EveJima/releases
+ Добавлены cookes в встроенный браузер. Теперь вам не нужно логинится на ваши сайты каждый раз при запуске EveJima
+ Добавлена система автообновления программы. Вы не обязаны обновлятся до последней версии. При выборе обновления может появится сообщение антивируса о том, что программа скачивает исполняемые файлы. Вы можете добавить программу в исключения, можете пойти на страницу релиза и взять новую версию, или продолжать использовать старую версию без обновления.

Исправленные ошибки:

+ Решена проблема с Windows 7 x64
+ Программе больше не требуется NET Framework 4.5.2 и она может использовать Framework 4.5

Patch notes for 1.28 Release 

Features & Changes:

+ Change the address of the repository from https://github.com/dunvit/WHL/releases to https://github.com/dunvit/EveJima/releases
+ Added built-in browser cookies. Now you do not need to login to your sites every time you start EveJima
+ Added auto-update system software. You are not required to be updated to the latest version. When choosing updates antivirus can appear a message that the software download executable files. You can add the program as an exception, you can go to the release page and get new version, or continue to use the old version without the upgrade.

Defect Fixes:

+ Resolves an issue with Windows  7 x64
+ Program no longer requires NET Framework 4.5.2 and it can use Framework 4.5
