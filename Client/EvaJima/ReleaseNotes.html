Version 2.05

Список изменений в обновлении 2.05


Дополнения и изменения:

  • Добавлена новая вкладка "История путешествий" для мониторинга сигнатур в системах.
    Для копирования всех космических сигнатур перейдите в клиент Eve Online, кликните левой кнопкой мыши на списке сигнатур. Выберите все (CRL+A) и скопируйте их в память (CRL-C). После этого вернитесь сюда и нажмите кнопку "Показать новые сигнатуры".

  • После авторизации пилотов автоматически открывается вкладка "Местоположение" для активного пилота.
  • Добавлен параметр позволяющий игнорировать цвета солнечных систем зависящие от уровня безопасности.

  • Добавлено имя активного пилота в заголовок окна.
  • В закладках "Информация" и "Местоположение" кнопка быстрой навигации к сайту "Superpute" заменена на сайт "eve-wh.space"

Исправленные ошибки:

  • Исправлен баг приводивший к переполнению памяти.
  • При отключении динамической карты EveJima перестает потреблять лишние ресурсы.

Version 2.04

Список изменений в обновлении 2.04


Дополнения и изменения:

  • EveJima полностью переведена на систему API ESI (В связи окончания поддержки ССР системы API CREST)
  • Добавлена возможность перехватывать ссылки из клиента Eve Online и открывать их в EveJima. Работает только для компьютеров с Chrome или Firefox браузерами по умолчанию. Эта опция по умолчанию выключина. Вы можете включить ее во вкладке "Установки". Предупреждение - включение этой опции производят временный изменения в Regestry и иногда негативно воспринимаются антивирусами.
  • Добавлена возможность открывать zkillboard при копировании имени пилота в клиенте Eve Online (можно копировать имя в списке чата установив на него фокус и нажав Crl+C)
  • Возвращена вкладка с возможностью задать автопилот для выбранного пилота. Данные о звездных системах берутся из файла.
  • Добавлена возможность открывать все информативные линки только в одном браузере EveJima.

Перехват линков из клиента Евы

  • Во вкладке "Установки" выберите "Открывать линки из клиента Евы" и перезапустите EveJima
  • В клиенте Евы во вкладке "Сбросить настройки" очистите внение ссылки.
  • EveJima перехватывает линки из Евы и открывает в своем встроенном браузере. Для того, чтобы перехватить линк EveJima меняет Registry в вашем компьютере перенаправляя все ссылки вместо Chrome или Firefox на файл лежащий в папке EveJima и называющийся "EveJimaBrowserInterceptor.exe"
  • Перехват линков происходит только если активно клиента Евы. Если вы закрываете Еву или табаетесь в любое другое окно EveJima возвращает в Registry обычный указатель на Chrome или Firefox.
  • Из-за игр с Registry некоторые антивирусы нервно реагируют на EveJima с конфигурацией перехвата линков. Добавьте файлы EveJima.exe и EveJimaBrowserInterceptor.exe в исключения, или откажитесь от опции перехвата линков.

Встроенный ZKillboard

  • EveJima открывает браузер с killboard пилота при выделении его ника в списке чата и нажатия CRL+C
  • EveJima открывает браузер с killboard пилота, корпорации или альянса при перетягивании названия в чат и нажатии CRL-C или CRL-X Наличие пробелов в начале или конце названия не влияют на результат.

 

Patch notes for 2.04 Release 

 

Features & Changes:

  • EveJima moved to use API ESI (ССР end support API CREST)
  • IGB option - now we open links from Eve Online in EveJima. Works only for default browsers Chrome or Firefox. This option is unselected. For enable this option use tab "Settings". EveJima change registry and some antivirus programs can issue warnings.
  • Added opportunity open zkillboard when copying the pilot name in the client Eve Online (You can copy the name in the chat list by setting focus to it and pressing Crl+C)
  • Added tab with the ability to set the autopilot for the selected pilot. Data on stellar systems are taken from the file.
  • Added ability to open all informative links only in one EveJima browser.
